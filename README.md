# ❄️ Fazha's Dotfiles

Selamat datang di repositori dotfiles saya! Ini adalah konfigurasi pribadi untuk sistem Arch Linux saya yang menggunakan Hyprland sebagai *Wayland Compositor*. Dikelola dengan sangat rapi menggunakan **GNU Stow**.

![Screenshot](https://via.placeholder.com/800x450.png?text=Taruh+Screenshot+Desktop+Kamu+Di+Sini)

## 💻 Spesifikasi Sistem

| Komponen | Pilihan |
| --- | --- |
| **OS** | [Arch Linux](https://archlinux.org/) |
| **WM/Compositor** | [Hyprland](https://hyprland.org/) |
| **Terminal** | [Kitty](https://sw.kovidgoyal.net/kitty/) |
| **Text Editor** | [Neovim](https://neovim.io/) (via [LazyVim](https://www.lazyvim.org/)) |
| **Shell** | Zsh |
| **Dotfiles Manager**| GNU Stow |
